# DevFocus 발표 원고

## 1. 서비스 소개

DevFocus는 강의 탐색, 결제, 학습 진도, 집중 타이머, Q&A와 코드 스니펫을
통합한 개발자 학습 플랫폼입니다. 학생뿐 아니라 강사와 운영자 업무까지 하나의
서비스에서 처리하도록 설계했습니다.

## 2. P1

Next.js 모바일 반응형 UI와 Express CRUD API를 구현했고 모든 핵심 데이터는
MySQL에 저장합니다. 강의, 레슨, 진도, 집중 기록, 질문, 답변, 댓글, 스니펫이
DB 관계와 외래키를 갖습니다.

## 3. P2

JWT 인증과 student, instructor, admin 역할을 적용했습니다. 강사는 본인 강의만
수정하며 관리자는 회원 역할, 전체 강의, 결제와 환불을 관리합니다. 프런트 권한
표시와 별개로 API에서 역할과 리소스 소유권을 다시 검사합니다.

## 4. P3

이메일과 Discord 알림, HMAC Webhook, 결제 만료·데이터 정리·정산 스케줄러,
JSON 로그, 요청 ID, Health와 Metrics를 구현했습니다. SQL 파라미터 바인딩,
Helmet, Rate Limit, 비밀키 분리로 보안을 강화했습니다.

## 5. 학생 수강 경험

결제 후 자동 수강 등록, 이어보기, 이전·다음 레슨, 30초 자동 진도 저장,
90% 완강 판정, 커리큘럼 완료 표시, 수강평과 개인 대시보드를 제공합니다.

## 6. 결제

토스페이먼츠 결제위젯을 사용합니다. 브라우저 결제 후 서버에서 orderId와 DB 가격을
검증하고 토스 승인 API를 호출합니다. 관리자 환불도 토스 취소 API와 연결됩니다.
실제 운영 결제는 가맹점 키와 운영 도메인 설정 후 활성화됩니다.

## 7. 강사와 관리자

강사는 강의·레슨 CRUD와 매출, 수강생, 학습률을 확인합니다. 관리자는 회원 역할,
강의, 결제·환불, 배치 로그, Webhook 처리 상태를 운영 콘솔에서 확인합니다.

## 8. 배포와 개선

Docker Compose와 Dockerfile, 환경변수 예제, 운영 Runbook을 준비했습니다. 현재는
공개 배포 전 상태이며, 다음 단계는 관리형 MySQL·API·Frontend 배포, 운영 secret
등록, 토스 운영 키 전환과 E2E 모니터링입니다.
